**Resonate - Open Source Real-Time Audio Communication Platform**

---

**Overview**

Resonate is an Open Source alternative to popular social voice platforms like Clubhouse and Twitter Spaces. It empowers the open-source community by providing a credible and user-friendly platform for real-time audio communication. With Resonate, users can join rooms, create and moderate events, engage in pair chatting, and soon enjoy real-time messaging.

---

**Features**

- **Real-Time Audio Communication**: Join rooms and talk to people in real-time.
- **Room Creation and Moderation**: Create rooms, moderate speakers, and organize events.
- **Pair Chatting**: Connect with random partners for engaging conversations.
- **Real-Time Messaging** *(Coming Soon)*: Stay connected with instant messaging.

---

**Installation Guide**

Follow these steps to set up Resonate locally on your system:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/AOSSIE-Org/Resonate-Website.git
   ```

2. **Navigate to the Project Directory**
   ```bash
   cd Resonate-Website
   ```

3. **Install Dependencies**
   ```bash
   npm i
   ```

4. **Start the Application**
   ```bash
   npm start
   ```
